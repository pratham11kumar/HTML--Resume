# HTML Resume Page

This project is a single page resume website built using only HTML.
It demonstrates clean structure, proper semantic tags, and a readable
layout without using any CSS or JavaScript.

## Features
- Semantic HTML structure
- Sections for About, Skills, Experience, Projects, Education and Contact
- Tables for structured information
- Simple and clean layout
- No external libraries or styling

## Setup Steps

1. Download or clone this repository.
2. Open the project folder.
3. Double click `index.html` to open it in your web browser.

No installation or extra software is required.

## Usage

- Open `index.html` in any browser to view the resume.
- Edit the `index.html` file to change your personal details, skills or projects.
- Refresh the browser to see your updates.
